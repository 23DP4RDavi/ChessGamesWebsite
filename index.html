Features

  • Play Chess and Checkers against another player.
  • Graphical Board: Visual board with custom pieces and timers.
  • Player Management: Add new players and view win statistics.
  • Save & Load: Save your game progress and resume later.
  • Main Menu: Easy navigation for starting games, loading, and managing players.

Getting Started

Requirements:

  • .NET 6.0 SDK or newer
  • Windows (for WinForms support)

How to Run:

  1. Clone the repository
  2. Build: dotnet build ChessGames/ConsoleApp1/ConsoleApp1.csproj
  3. Run: dotnet run --project ChessGames/ConsoleApp1/ConsoleApp1.csproj
